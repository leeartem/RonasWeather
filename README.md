# Ronas Weather

Простейший бэкенд (по сути обертка для внешнего API) в рамках тестового задания.

## Команды

Запуск контейнера:
> make run

В случае проблем с правами:
> make optimize

Запуск тестов:
> make test

Остановить контейнер:
> make stop


## Куда смотреть
Работает на 80 порту:
> http://localhost/
Там же на главной будет описание ендпоинтов.
